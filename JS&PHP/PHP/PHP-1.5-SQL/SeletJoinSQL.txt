La liste de tous les utilisateurs associés à leurs villes respectives
SELECT first_name, last_name, street, number, complements, postal_code, city FROM `users`, `addresses` WHERE users.address_id = addresses.id

La liste de tous les utilisateurs vivant à une adresse sans numéro
SELECT first_name, last_name, street, number, complements, postal_code, city FROM `users`, `addresses` WHERE users.address_id = addresses.id AND addresses.number IS NULL

La liste de tous les produits dont le prix est supérieur à 1 000 €
SELECT * FROM `products` WHERE price > 1000

La liste de tous les produits associés à leurs photos respectives
SELECT name, description, price, url FROM `pictures`, `products` WHERE pictures.product_id = products.id 

La liste de tous les produits appartenant à la catégorie « Voyage »
SELECT name, products.description, price, title, categories.description FROM `products`,`categories` WHERE categories.title = 'Voyage'

La liste de tous les utilisateurs ayant effectué plus de dix commandes
SELECT first_name, last_name FROM users u
INNER JOIN orders o ON u.id = o.user_id
HAVING COUNT(o.user_id) > 10

La liste de tous les produits achetés par le premier utilisateur inscrit.
